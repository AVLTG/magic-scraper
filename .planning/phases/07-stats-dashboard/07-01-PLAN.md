   - Import `type { Game } from '@/app/games/page'` (or define a local compatible type)
   - Export `isoWeekStartUTC(isoDateString: string): string` — returns YYYY-MM-DD of the Monday of the ISO week containing the date, using `getUTCDay()` (NOT `getDay()`) per D-21
   - Export `weeksBetween(startWeek: string, endWeek: string): string[]` — returns all Monday YYYY-MM-DD strings between start and end inclusive, stepping 7 days
   - Export `computePlayerWinRate(games: Game[]): { player: string; wins: number; played: number; rate: number }[]` — includes ALL games (imported included per D-17), omits players with played===0 (D-19), sorted by rate descending
   - Export `computeDeckWinRate(games: Game[]): { deck: string; wins: number; played: number; rate: number }[]` — EXCLUDES imported games (`games.filter(g => !g.isImported)`) per D-16, omits decks with played===0 (D-19), sorted by rate descending
   - Export `computeScrewedRate(games: Game[]): { player: string; screwed: number; played: number; rate: number }[]` — includes ALL games (D-17), omits players with played===0, sorted by rate descending
   - Export `computeWeeklyFrequency(games: Game[]): { weekStart: string; gameCount: number }[]` — includes ALL games (D-17), buckets by isoWeekStartUTC, fills all weeks between min and max with 0 for empty weeks (D-22a), sorted chronologically
   - Export `computeMostLikelyToPlay(games: Game[]): { player: string; participations: number; totalGames: number; rate: number }[]` — formula is participations/totalGames (D-23), includes ALL games (D-17), omits players with participations===0, sorted by rate descending
   - Export `computeMostLikelyToPlayBump(games: Game[]): { weekStart: string; ranks: { player: string; rank: number }[] }[]` — cumulative participation rate through each week, rank 1 = highest rate, ties share rank (D-25), includes ALL games (D-17), sorted chronologically
   - Export `computeWinsByPlayerPie(games: Game[]): { player: string; wins: number }[]` — includes ALL games (D-17), omits players with wins===0, sorted by wins descending
   - Export `computeGamesByDeckPie(games: Game[]): { deck: string; games: number }[]` — EXCLUDES imported games per D-16, omits decks with games===0, sorted by games descending
   - Export `computePlayerRadar(games: Game[]): { player: string; played: number; wins: number; screwed: number; wonByCombo: number }[]` — `played`/`wins`/`screwed` include ALL games (D-27), `wonByCombo` counts wins where `game.wonByCombo === true && !game.isImported` (D-27), omits players with played===0

   Key implementation details:
   - `deckName` can be null on participants; for deck stats, skip participants where `deckName` is null or empty string
   - For `computeDeckWinRate`: a game's winner deck is the `deckName` of the participant where `isWinner === true`; the denominator "played" for a deck is the count of games where ANY participant used that deck (not just winner uses)
   - For `computeGamesByDeckPie`: count total participant appearances with each deck name (how many times a deck was played, not how many games it won)
   - Rate values are 0..1 floats (not percentages)
