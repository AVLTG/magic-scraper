<objective>
Replace the full `puppeteer` package with `@sparticuz/chromium-min` + `puppeteer-core` for LGS scrapers, and replace the Moxfield browser scraper with a plain `fetch()` call. After this plan, no code path launches the full bundled Puppeteer browser.

Purpose: The full `puppeteer` package bundles a ~300 MB Chromium binary, exceeding Vercel's 250 MB function limit. `chromium-min` downloads the binary at runtime to `/tmp`, and `puppeteer-core` connects to it without bundling anything. The Moxfield scraper already hits a JSON API via `page.goto()` -- replacing with `fetch()` eliminates browser usage entirely for that path.

Output: All scrapers work against the new packages; `npm run build` succeeds; no import of `puppeteer` (full package) remains anywhere.
</objective>

Replace the entire file:

```typescript
import "server-only";
import { launchBrowser } from "./browser";
import { scrapeETB } from "./scrapeETB";
import { scrapeDCC } from "./scrapeDCC";
import { scrapeFTF } from "./scrapeFTF";
import type { Product } from "@/types/product";

export async function scrapeAllSites(
  card: string
): Promise<{ products: Product[]; failedStores: string[] }> {
  const browser = await launchBrowser();
  try {
    const results = await Promise.allSettled([
      scrapeETB({ card, browser }),
      scrapeDCC({ card, browser }),
      scrapeFTF({ card, browser }),
    ]);

    const products: Product[] = [];
    const failedStores: string[] = [];
    const storeNames = [
      "Enter The Battlefield",
      "Dungeon Comics and Cards",
      "Face to Face Games",
    ];

    results.forEach((result, i) => {
      if (result.status === "fulfilled") {
        products.push(...result.value);
      } else {
        console.error(`${storeNames[i]} failed:`, result.reason);
        failedStores.push(storeNames[i]);
      }
    });

    return { products, failedStores };
  } finally {
    await browser.close();
  }
}
```

```typescript
import "server-only";
import type { MoxfieldCard } from "@/types/moxfield";

const USER_AGENT =
  "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36";

export async function scrapeMoxfield({
  collectionId,
}: {
  collectionId: string;
}): Promise<MoxfieldCard[]> {
  console.log("Starting scrape for collection:", collectionId);

  let allCards: MoxfieldCard[] = [];
  let pageNumber = 1;
  let hasMorePages = true;
  const pageSize = 5000;

  while (hasMorePages) {
    const apiUrl = `https://api2.moxfield.com/v1/collections/search/${collectionId}?sortType=cardName&sortDirection=ascending&pageNumber=${pageNumber}&pageSize=${pageSize}&playStyle=paperDollars&pricingProvider=cardkingdom`;
    console.log(`Fetching page ${pageNumber}...`);

    const response = await fetch(apiUrl, {
      headers: { "User-Agent": USER_AGENT },
    });

    if (!response.ok) {
      console.error(`Moxfield API returned ${response.status}`);
      break;
    }

    const apiData = await response.json();

    if (!apiData || !apiData.data) {
      break;
    }

    const pageCards: MoxfieldCard[] = [];

    for (const key of Object.keys(apiData.data)) {
      const item = apiData.data[key];

      if (!item || !item.card) {
        continue;
      }

      const isBasicLand = item.card.type_line?.startsWith("Basic Land");
      const isToken = item.card.type_line?.includes("Token");

      if (isBasicLand || isToken) {
        continue;
      }

      pageCards.push({
        name: item.card.name,
        scryfall_id: item.card.scryfall_id,
        quantity: item.quantity,
        condition: item.condition,
        isFoil: item.isFoil,
        set: item.card.set,
        set_name: item.card.set_name,
        type_line: item.card.type_line || "",
      });
    }

    console.log(`  Got ${pageCards.length} cards from page ${pageNumber}`);
    allCards = allCards.concat(pageCards);

    if (pageCards.length < pageSize || Object.keys(apiData.data).length === 0) {
      hasMorePages = false;
    } else {
      pageNumber++;
    }
  }

  console.log("Total cards processed:", allCards.length);
  return allCards;
}
```

Key differences from original:
- Removed: `import puppeteer from 'puppeteer';`
- Removed: All `browser` / `page` creation and cleanup
- Added: `fetch()` with `User-Agent` header instead of `page.goto()` + `page.evaluate()`
- Added: `!response.ok` check instead of relying on puppeteer timeout
- Kept: Identical pagination logic, card filtering (skip basic lands + tokens), MoxfieldCard[] return type
- Kept: `"server-only"` import
